# ojs-beautifier

Custom CSS to enhance the look and feel of OJS journals.

## Overview

`ojs-beautifier` provides a collection of custom CSS styles to improve the appearance, readability, and layout of journals powered by [Open Journal Systems (OJS)](https://pkp.sfu.ca/ojs/). This repository is ideal for journal editors and web managers who want to quickly enhance their journal’s design without modifying the core OJS files.

## Features

- Modern, clean typography and layout adjustments  
- Improved readability for articles and abstracts  
- Custom styling for headers, footers, and menus  
- Responsive design improvements for mobile and tablet views  
- Easy to integrate with any OJS theme  

## Installation

1. Download the CSS file(s) from this repository.  
2. Upload them to your OJS theme folder or include via the **Additional CSS** option in your journal’s theme settings.  
3. Clear the OJS cache and refresh your site to see the changes.  

## Usage

- You can include the CSS file directly in your theme’s `styles.css` or via `<link>` in your header template.  
- Customize colors, fonts, or spacing by editing the CSS as needed.
